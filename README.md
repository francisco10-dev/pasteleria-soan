# 🎂 Pastelería Soan – Landing Page

Landing page para **Pastelería Soan**, emprendimiento de pastelería artesanal ubicado en Santa Cruz, Guanacaste, Costa Rica.

## Stack

- **React 18** + **TypeScript**
- **Vite 5**
- **CSS puro** (sin frameworks CSS) – Mobile First
- **Fuente**: Maven Pro (Google Fonts)

## Instalación y uso

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview
```
